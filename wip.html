RStudio Tutorials

This collection of RStudio Tutorials was designed for absolute beginners to RStudio who have a background in basic behavioral statistics or are enrolled in an introductory statistics course. This material is not intended to teach statistical concepts, rather, the tutorials show how to use RStudio to apply common statistical techniques to behavioral data.

  • Tutorial 0: Getting Started with RStudio & RMarkdown

    Learn how to install RStudio, run functions in the console, install new packages, and work in RMarkdown

  • Tutorial 1: Descriptive Statistics & Data Visualization

    Learn how to import and save data, calculate descriptive statistics, and visualize relationships

  • Tutorial 2: Distributions & Hypothesis Testing

    Learn how to simulate a distribution of scores and perform a null-hypothesis test using the one-sample t-statistic

  • Tutorial 3: t-Statistic

    Learn how to visualize relationships, conduct a null hypothesis test using the t-statistic, and find effect size

  • Tutorial 4: One-Factor ANOVA

    Learn how to conduct a one-factor analysis of variance, run a Tukey's HSD test, and find effect size

  • Tutorial 5: Multiple Regression

    Learn how to create scatter plots, conduct an analysis of multiple regression, and test its assumptions

  • Demo: Survey Data Analysis

    An introduction to analyzing survey research data and communicating your findings
